# Price Tracker

Price tracker Api.

## Project setup

```bash
1. create a .env file in root of project folder and copy the environment variables from .en.example file 
2. npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run via Docker
```bash
1. create a .env file in root of project folder and copy the environment variables from .en.example file 
2. docker-compose up
```
